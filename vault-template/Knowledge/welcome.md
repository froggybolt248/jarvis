# Welcome to your vault

This folder is your knowledge bank. Every note is a plain markdown file that you
own — you can open this folder in Obsidian, sync it with git, or edit files in
any text editor. Jarvis indexes everything here and cites these files when it
answers your questions.

A few things worth knowing:

- **Daily notes** land in `Daily/` — one file per day with your briefing, meals,
  workouts, and what Jarvis did for you.
- **Core memory** lives in `00-Core/core-memory.md` — the facts Jarvis always
  keeps in mind. You can edit it anytime.
- Nothing in this vault ever leaves your machine.

Try asking Jarvis: *"What do you know about my vault?"* — it should cite this
very file.
